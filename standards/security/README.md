# security

Gold-standard cross-cutting rubric for application security across browser, server,
edge, CLI, SDK, MCP, and CI/deployment boundaries.

Published rubric:
<https://vibecodeqa.online/standards/security/v1/>

## What this standard is

This standard does not replace OWASP ASVS or the OWASP Cheat Sheet Series. It cites those
authorities, then defines the VCQA-owned signals a repo judge should inspect when
security crosses stack boundaries.
