# cloudflare-pages-fullstack

The **gold-standard reference** for the `cloudflare-pages-fullstack` archetype: a static
frontend co-deployed with Cloudflare Pages Functions, usually exposing same-origin
`/api/*` routes.

Published by VibeCode QA at:

<https://vibecodeqa.online/standards/cloudflare-pages-fullstack/>

## What this is

This repo is a rubric, not a tutorial. VibeCode QA detects a repo's stack shape, loads the
matching standard, and scores the code against stable rule IDs.

This standard composes:

- React SPA/static frontend expectations
- Cloudflare Pages Functions routing and middleware
- TypeScript contracts
- web security
- GitHub Actions / Cloudflare Pages deployment
