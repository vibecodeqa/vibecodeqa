6 categories of analysis

Every dimension of code quality, scored 0-100 with actionable findings

Foundations

23% of score

Project structure, lint, TypeScript types, type safety (as any counts), code standards (naming, file size, code smells).

Quality

15% of score

Cognitive complexity per function, code duplication detection, documentation coverage (README + JSDoc).

Testing

22% of score

Testing pyramid (unit/integration/component/E2E), coverage, file pairing, assertion density, mock ratio, snapshot smell.

Architecture

7% of score

Import graph analysis, circular dependencies, god modules, orphan files, fan-out coupling. SVG diagram included.

Security

18% of score

Secret detection (13 patterns), 15 CWE-mapped vulnerability patterns, dependency audit + outdated checks.

LLM Readiness

15% of score

Confusion Index (naming ambiguity that causes AI to edit wrong code) + Context Locality (file self-containment for LLM comprehension). Research-backed, unique to VibeCode QA.
